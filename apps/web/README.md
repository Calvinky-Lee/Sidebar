# @sidebar/web

Frontend for The Sidebar (spec 07). Next.js App Router + Tailwind + Framer Motion.

```
pnpm dev:web      # from the repo root
```

Home page is a working first slice (search bar, intake, orb field). The radial
HQ, ThinkBubbles, decision theater, and dev-replay harness described in
`tasks/P3-frontend.md` are not built yet — see routing stubs under `app/session`,
`app/replay`, and `app/dev/replay`.
